# 测试用例（云作品小程序）

## 1. 页面加载
- 打开小程序，进入「我的作品」页面
- 页面显示加载中，随后加载作品列表（若无作品则显示空列表）
- 检查请求 `listWorks` 云函数返回 `errCode: 0`

## 2. 作品列表渲染
- 作品卡片按创建时间倒序排列
- 封面图正确显示，标签与数量文本与数据库一致
- 渐变卡片背景色与标签样式符合设计

## 3. 交互占位
- 点击卡片：Toast 显示「点击卡片 +id」
- 点击「新建作品」：Toast 显示「新建作品（占位）」
- 点击「搜索」：Toast 显示「搜索（占位）」

## 4. 错误处理
- 断网或云函数异常：显示错误提示与「重试」按钮
- 重试可再次调用 `listWorks`

## 5. 云函数测试（在云开发控制台）
- `listWorks`：分页、标签过滤
- `createWork`：必填参数校验（title、imageFileIds）
- `updateWork`：仅本人可更新
- `deleteWork`：级联删除云存储文件

## 6. 权限测试
- 切换登录用户，不可看到他人作品
- 尝试修改/删除他人作品应返回 404
